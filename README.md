# 逻辑流程
## 维修与维护的关系

法国人认为

- 如果现在处于维护中，那么即使维修时间插在中间，下一个事件还是维护结束
- 维护**结束**后刷新维修
- 维修维护互换同理
- 交货可以插在维修或维护中间，需要加if分支

这样和我们自己的逻辑等效

- 维护**开始**时就刷新维修
- 维修刷新时间要加上一个维护的长度
- 维修维护互换同理
- 交货独立于维修或维护计算
