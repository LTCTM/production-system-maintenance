# 多机器

所有多机器理论，法国人都只考虑两机器

状态切换理论：同时只能触发一个事件。一般而言，两个机器的状态不可能同时发生变化。但也有例外：若并行生产，两个机器都是满仓状态。交货时，它们都变为生产中，此时它们的状态同时发生变化

双机器事件下标名称：第1位是事件名，第2位是机器名

# 并行生产 parallèle

- 只考虑两机器并行，虽然代码上两机器和多机器是一样的

# 串行生产 série

![](img/串联.jpg)

机器1生产半成品，放进箱子1，机器2从箱子1拿半成品，生产成品，放进箱子2。每1个半成品可以生产1个成品

每个生产环节都有彼此独立的常量
